# InsureFlow Help Navigator — Knowledge Base

Drop your FAQ and policy documents here. The Help Navigator reads every file in this folder to answer broker questions.

## Supported formats
- `.md` — Markdown (recommended)
- `.txt` — Plain text

## Tips
- Use clear headings to separate topics (e.g. `## Coverage Questions`)
- One file per topic is easier to maintain (e.g. `coverage-faq.md`, `claims-process.md`)
- If an answer is not found in any file, the bot says it is out of scope

## Example structure
```
knowledge/
  general-faq.md          ← general product questions
  coverage-faq.md         ← what is/isn't covered
  claims-process.md       ← how to file a claim
  underwriting-rules.md   ← eligibility & decline reasons
  pricing-guide.md        ← how premiums are calculated
```
